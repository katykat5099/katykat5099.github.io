ALPHA WORLD

"ALPHA WORLD" is a new Minecraft Server project started on April 18th, 2024.

The goal is to recreate the history of Minecraft updates through scheduled upgrades of the server.

Through this webpage you will find said schedule, and any useful information to help you out in the server.

Current Version:

This project started in version Alpha 1.1.2_01

The reason it did not start any earlier is due to bugs with older versions of the server that make progress not very easy. Starting on this version is the friendliest possible approach for everyone.
