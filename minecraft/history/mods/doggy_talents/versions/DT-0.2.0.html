Doggy Talents v0.2.0

Mobs in this Version:

Wolves

Tamed Wolves can be upgraded into 3 different upgraded variants, using 3 different items. The chance is about 1/16 to succeed.

Fighter Wolf

A tamed wolf upgrade using Leather. Does about double the damage of a normal tamed wolf.

Creeper Sweeper

A tamed wolf upgrade using Gunpowder. Reacts to nearby creepers.

GlownWolf

A tamed wolf upgrade using Glowstone. Drops 4 torches when fed at full health.
